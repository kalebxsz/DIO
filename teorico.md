# Projeto do desafio de projeto sobre Git/GIThub da DIO 

desafio de projeto sobre Git/Github 

Aqui esta o passo a passo que utilizei para criar meu primeiro repositório. 


1 - criei uma pasta no meu computador local 

2 - criei o arquivo teorico.md

3 - inicializei o repositório com git init 

4 - criei 3 pastas com os nomes(html, css e javascript)

5 - adicionei todo o conteúdo com          ==>  git add . <==

6 - criei um commit com                    ==> git commit -m "Primeiro commit sobre o curso"  <==

7 - atrelei tudo isso a uma branch com     ==> git branch -M main  <==

8 - adicionei o meu repositório remoto com ==> git remote add origin git@github.com:kalebxsz/DIO.git

9 - Enviei o meu projeto para o GitHub com ==> git push -u origin main <==



Esse passo a passo foi o qual eu me acostumei, após aprender com o curso e conteudo externo.
